# User Manual

## Start
1. Open the web page (`index.html`) via a static server.
2. Enter player name.
3. Select difficulty and click **Start Battle**.

## Controls
- Move: `W` `A` `S` `D` or arrow keys
- Shoot: `Space`
- Return to menu: `R`

## Game Rules
- 5 levels total.
- You have 3 lives.
- After respawn you are briefly invincible.
- Brick walls can be destroyed.
- Steel walls block bullets.

## Ranking
- Ranking is per difficulty.
- Stored in browser IndexedDB.
- Sort order:
  1. Rating DESC
  2. Kills DESC
  3. Time ASC
